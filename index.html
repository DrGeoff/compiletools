cake

Build C++ fast, with practically no configuration.

Authors

Zomojo Pty, Ltd. - Matthew Herrmann (matthewinrandwick@gmail.com)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/matthewinrandwick/Cake

Getting Started

To use cake, download the source code: 

    $ cd <unzipped-location>
    $ sudo cp cake /usr/bin/cake
    $ sudo cp cake /etc/cake
